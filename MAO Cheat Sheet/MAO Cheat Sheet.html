Web Development Cheat Sheet

Here you can find the most important codes and tips about HTML, CSS and JavaScript.
HTML Cheat Sheet
Element Syntax Description
h1 <h1>content</h1> Defines the most important heading. There are six levels of headings from <h1> (most important) to <h6> (least important). Used to indicate main headings in a document.
p <p>content</p> Defines a paragraph. It is a block-level element and typically contains text content, including other inline elements like links or emphasized text.
div <div>content</div> Defines a division or section in an HTML document. It is a block-level element commonly used to group content for styling or layout purposes using CSS.
a <a href="url">link</a> Defines a hyperlink, which is used to link from one page to another. The "href" attribute specifies the URL of the page the link goes to. It can also link to other resources, such as email addresses or file downloads.
img <img src="url" alt="description"> Embeds an image in the document. The "src" attribute specifies the path to the image file, and the "alt" attribute provides alternative text for screen readers and if the image fails to load.
ul <ul><li>item</li></ul> Defines an unordered (bulleted) list. The <ul> element contains one or more <li> (list item) elements.
ol <ol><li>item</li></ol> Defines an ordered (numbered) list. The <ol> element contains one or more <li> (list item) elements.
li <li>item</li> Defines a list item. It is used within <ul> (unordered list) or <ol> (ordered list) elements.
table <table>content</table> Defines a table. It contains rows (<tr>), and each row contains cells (<td> for data cells and <th> for header cells).
tr <tr>content</tr> Defines a row in a table. It contains one or more <td> (data cell) or <th> (header cell) elements.
td <td>content</td> Defines a cell in a table. It is used within a <tr> (table row) element to contain data.
th <th>content</th> Defines a header cell in a table. It is used within a <tr> (table row) element and usually appears at the top of a column or at the start of a row. By default, header cells are bold and centered.
form <form>content</form> Defines an HTML form for user input. The form element can contain various input elements like text fields, checkboxes, radio buttons, submit buttons, etc. The "action" attribute specifies where to send the form data when it is submitted.
input <input type="text"> Defines an input control. It is used within a form to accept user input. The "type" attribute specifies the type of input (e.g., text, password, email, etc.).
button <button>content</button> Defines a clickable button. It can be used within or outside a form to trigger actions defined in JavaScript or to submit form data.
br <br> Inserts a single line break. It is an empty element, meaning it does not have a closing tag. It is used to start a new line of text without starting a new paragraph.
hr <hr> Inserts a thematic break (horizontal rule). It is an empty element and is typically used to separate content or sections within a page.
span <span>content</span> Defines a section in a document. It is an inline element used to group inline elements or text for styling purposes using CSS.
strong <strong>content</strong> Defines important text. It renders as bold text and conveys that the enclosed text is of strong importance.
em <em>content</em> Defines emphasized text. It renders as italicized text and conveys emphasis on the enclosed text.
HTML-Semantic Cheat Sheet
Tag Syntax Description
header <header>content</header> Defines a header section for a document or a section. Typically contains introductory content or navigational links. It can contain heading elements, logo, search form, etc.
nav <nav>content</nav> Defines a section of navigation links. Used for primary navigation menus. It helps search engines and assistive technologies to understand the structure of your site.
section <section>content</section> Defines a section in a document. Used to group related content and typically contains a heading. Suitable for thematic grouping of content.
article <article>content</article> Defines an independent, self-contained piece of content. Suitable for blog posts, news articles, or other content that could stand alone.
aside <aside>content</aside> Defines content aside from the main content. Typically used for sidebars, pull quotes, or groups of related content.
footer <footer>content</footer> Defines a footer for a document or a section. Typically contains information about the author, copyright information, or links to related documents.
main <main>content</main> Defines the main content of a document. It should contain content that is directly related to the central topic of the document, excluding headers, footers, navigation, and sidebars.
figure <figure>content</figure> Defines self-contained content, such as illustrations, diagrams, photos, code listings, etc. It can contain a <figcaption> element to provide a caption for the content.
figcaption <figcaption>content</figcaption> Defines a caption for a <figure> element. It is used to provide a title or explanatory text for the content in the figure.
mark <mark>content</mark> Defines text that should be marked or highlighted. It is used to draw attention to specific text within a larger body of text.
time <time>content</time> Defines a specific time (or a datetime). It is used to encode dates and times in a machine-readable format, useful for adding times and dates to your document that computers can recognize and manipulate.
address <address>content</address> Defines contact information for the author or owner of a document or article. Typically includes physical addresses, email addresses, phone numbers, etc.
progress <progress>content</progress> Represents the completion progress of a task. It is typically displayed as a progress bar.
details <details>content</details> Defines additional details that the user can view or hide. It is often used with the <summary> element, which provides a summary or label for the details.
summary <summary>content</summary> Defines a visible heading for a <details> element. When the summary is clicked, the details are shown or hidden.
output <output>content</output> Represents the result of a calculation or user action. Often used in conjunction with form elements.
CSS Cheat Sheet
Property Usage Description
color color: value; Sets the color of the text. The value can be a color name, hex code, rgb, rgba, hsl, or hsla.
background-color background-color: value; Sets the background color of an element. The value can be a color name, hex code, rgb, rgba, hsl, or hsla.
font-size font-size: value; Sets the size of the text. The value can be in pixels (px), em, rem, percentages, etc.
margin margin: value; Sets the outer spacing of an element. The value can be in pixels (px), em, rem, percentages, or auto.
padding padding: value; Sets the inner spacing of an element. The value can be in pixels (px), em, rem, or percentages.
border border: width style color; Sets the border of an element. The width is the thickness, the style can be solid, dotted, dashed, etc., and the color can be a color name, hex code, rgb, rgba, hsl, or hsla.
width width: value; Sets the width of an element. The value can be in pixels (px), em, rem, percentages, etc.
height height: value; Sets the height of an element. The value can be in pixels (px), em, rem, percentages, etc.
display display: value; Specifies how an element is displayed. The value can be block, inline, inline-block, flex, grid, none, etc.
position position: value; Specifies the positioning method. The value can be static, relative, absolute, fixed, or sticky.
top top: value; Specifies the top position of an element. Used with position: relative, absolute, fixed, or sticky. The value can be in pixels (px), em, rem, percentages, etc.
left left: value; Specifies the left position of an element. Used with position: relative, absolute, fixed, or sticky. The value can be in pixels (px), em, rem, percentages, etc.
right right: value; Specifies the right position of an element. Used with position: relative, absolute, fixed, or sticky. The value can be in pixels (px), em, rem, percentages, etc.
bottom bottom: value; Specifies the bottom position of an element. Used with position: relative, absolute, fixed, or sticky. The value can be in pixels (px), em, rem, percentages, etc.
flex flex: value; A shorthand property for flex-grow, flex-shrink, and flex-basis. It sets how a flex item will grow or shrink to fit the space.
grid-template-columns grid-template-columns: value; Specifies the columns of a grid container. The value can be a space-separated list of values, each representing the size of a column.
align-items align-items: value; Specifies the alignment of items inside a flex or grid container along the cross axis. The value can be flex-start, flex-end, center, baseline, or stretch.
justify-content justify-content: value; Specifies the alignment of items inside a flex or grid container along the main axis. The value can be flex-start, flex-end, center, space-between, space-around, or space-evenly.
text-align text-align: value; Sets the horizontal alignment of text. The value can be left, right, center, or justify.
font-family font-family: value; Specifies the font family for text. The value can be a specific font name, a generic family (like serif, sans-serif, etc.), or a comma-separated list of font names.
line-height line-height: value; Sets the height of a line of text. The value can be a number, a length (px, em, rem, etc.), or a percentage.
opacity opacity: value; Sets the opacity level of an element. The value can be a number between 0 (completely transparent) and 1 (completely opaque).
JavaScript Cheat Sheet
Method/Function Usage Description
alert() alert("message"); Displays an alert box with a specified message and an OK button.
console.log() console.log("message"); Prints a message to the browser console for debugging purposes.
document.getElementById() document.getElementById("id"); Returns the element that has the ID attribute with the specified value.
addEventListener() element.addEventListener("event", function); Attaches an event handler to the specified element.
querySelector() document.querySelector("selector"); Returns the first element that matches a specified CSS selector(s) in the document.
setTimeout() setTimeout(function, milliseconds); Calls a function or evaluates an expression after a specified number of milliseconds.
setInterval() setInterval(function, milliseconds); Calls a function or evaluates an expression at specified intervals (in milliseconds).
JSON.parse() JSON.parse("jsonString"); Parses a JSON string and returns the JavaScript object.
JSON.stringify() JSON.stringify(object); Converts a JavaScript object to a JSON string.
map() array.map(function); Creates a new array with the results of calling a provided function on every element in the calling array.
filter() array.filter(function); Creates a new array with all elements that pass the test implemented by the provided function.
reduce() array.reduce(function, initialValue); Executes a reducer function on each element of the array, resulting in a single output value.
forEach() array.forEach(function); Executes a provided function once for each array element.
push() array.push(element); Adds one or more elements to the end of an array and returns the new length of the array.
pop() array.pop(); Removes the last element from an array and returns that element.
shift() array.shift(); Removes the first element from an array and returns that element.
unshift() array.unshift(element); Adds one or more elements to the beginning of an array and returns the new length of the array.
find() array.find(function); Returns the value of the first element in the array that satisfies the provided testing function.
includes() array.includes(element); Determines whether an array includes a certain value among its entries, returning true or false as appropriate.
slice() array.slice(start, end); Returns a shallow copy of a portion of an array into a new array object selected from start to end (end not included).
splice() array.splice(start, deleteCount, item1, item2, ...); Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.
sort() array.sort(function); Sorts the elements of an array in place and returns the array. The default sort order is according to string Unicode code points.
